# YurikaCC - A C4 Repetition

YurikaCC 是基于 [C4](https://github.com/rswier/c4)（C in Four Functions）项目的一个复刻与学习版本。它是一个用 C 语言编写的极简 C 语言解释器/编译器，旨在探索自举和编译原理的核心思想。

## ✨ 已实现特性

- **完整的编译-解释流程**：支持词法分析、语法分析、代码生成和虚拟机执行。
- **基础数据类型**：支持 `int`、`char` 及多级指针类型。
- **控制流语句**：支持 `if` / `else` 分支和 `while` 循环。
- **函数定义与调用**：支持带参数的函数定义及递归调用。
- **系统调用封装**：内置了 `printf`、`malloc`、`memset` 等常用库函数的接口。
- **自举潜力**：保留了 C4 原有的自举设计思想。

## ⚠️ 重要限制与差异

本项目作为对 C4 的忠实复刻，保留了其极简设计带来的以下限制，使用时需特别注意：

- **禁止跨函数递归**  
  函数调用机制存在限制，**不支持函数 A 调用函数 B，再由函数 B 回调函数 A** 的间接递归模式。直接的函数自递归（如斐波那契）则正常工作。

- **不支持宏定义**  
  词法分析器会跳过所有以 `#` 开头的行，因此无法使用 `#include`、`#define` 等预处理指令。所有代码需写在同一源文件中。

- **局部变量必须最先声明**  
  在函数体内，所有局部变量的声明**必须出现在任何可执行语句之前**。混合声明与语句将导致解析错误。

- **类型检查宽松**  
  类型系统较为简化，某些隐式类型转换可能不会触发错误，但会导致未定义行为。

- **错误处理严格**  
  一旦遇到语法或语义错误，编译器将立即打印错误信息并退出。
